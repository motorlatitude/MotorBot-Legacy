Sort Current User's Playlists
PATCH
Endpoint


PATCH https://motorbot.io/api/user/sort/{playlist_id}/{position} HTTP/1.1
Authorization: Bearer {access_token}

Allows altering the order of the current users followed and created playlists.

Request Parameters

Path Parameters

Path Parameter Value
playlist_id id of the playlist of which the position is to be altered
position An integer describing the relative position of the playlist for the current user

Header Fields

Header Field Value
Authorization
REQUIRED		 A valid access token from the MotorBot authorization system. See the OAuth 2.0 endpoint or the User Authorization Guide for further information. The access token must have been issued on behalf of the current user.

Response Format

On success, the HTTP status code in the response header will be set to 204 No Content. Should an error occur, the header status code in the response header will be set to the corresponding error. If either your API key or the required access token for the requested user are not present a 401 Unauthorized response header will be returned.
