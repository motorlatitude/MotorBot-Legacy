Playlist

Endpoints for retrieving and altering playlists


Method Endpoint Usage Returns
GET /playlist/{playlist_id} Get Playlist playlist
POST /playlist/ Create Playlist playlist
DELETE /playlist/{playlist_id} Delete Playlist
PUT /{playlist_id}/song Import Song From Source
PATCH /{playlist_id}/song/{song_id} Import Song From MotorBot
DELETE /{playlist_id}/song/{song_id} Delete Song From Playlist
