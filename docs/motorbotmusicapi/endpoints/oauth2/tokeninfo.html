Get Access Token Information
GET
Endpoint


GET https://motorbot.io/api/oauth2/tokeninfo HTTP/1.1

Get information on an issued access_token.

Request Parameters

Query Parameters

Query Parameter Value
token
REQUIRED		 The issued access_token to receive information for

Response Format

On success, the HTTP status code in the response header will be set to 200 OK and the response body will contain an access token information object in JSON format. If the supplied token does not exist in our system a 404 Not Found will be returned. 

HTTP/1.1 200 OK
Content-Type: application/json

Access Token Information Object

Key Value
value The user issued access_token to be used in endpoints that require the Authorization request header
clientId The client_id of the app for which the access_token was generated for
userId The id of the user for which the access_token was issued on behalf of
expires The UNIX timestamp in milliseconds at which point the access_token will expire. By default the generated access_token is valid for 24 hours from the time issued
valid Is this access_token still valid to use, i.e. has it not expired or been revoked yet? 
{
    "value": {access_token},
    "clientId": {client_id},
    "userId": {user_id},
    "expires": 1484336450243,
    "valid": true
}
