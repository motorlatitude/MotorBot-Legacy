Import Spotify Playlist
PUT
Endpoint


PUT https://motorbot.io/api/spotify/{spotify_playlist_id}/owner/{spotify_owner_id} HTTP/1.1
Authorization: Bearer {access_token}

Import a spotify playlist into the authorised users list of playlists

Request Parameters

Path Parameters

Path Parameters Value
spotify_playlist_id
REQUIRED		 The spotify playlist_id of the spotify playlist to be imported
spotify_owner_id
REQUIRED		 The spotify owner_id, normally the creators username, of the spotify playlist to be imported

Header Fields

Header Field Value
Authorization
REQUIRED		 A valid access token from the MotorBot authorization system. See the OAuth 2.0 endpoint or the User Authorization Guide for further information. The access token must have been issued on behalf of the current user.

Response Format

On success, the HTTP status code in the response header will be set to 200 OK and the response body will contain a playlist object with Content-Type: application/json. Should an error occur, the header status code in the response header will be set to the corresponding error. If either your API key or the required access token for the requested user are not present a 401 Unauthorized response header will be returned.

This request will last longer than expected and will only return a response once completed, for larger playlists this request can take a few minutes. A status update event can be received through the MotorBot websocket connection. 

HTTP/1.1 200 OK
Content-Type: application/json

Playlist Import Object

Key Value Type Value Description
playlist Playlist Object Simplified playlist object of the created playlist
not_found object A list of song ids that were not successfully found and hence could not be imported from spotify 

{
  "playlist": {
    playlist object
  },
  "not_found": {
    not found object
  }
}

Playlist Object (Simplified)

Key Value Type Value Description
id string The unique identifier that is assigned to the playlist
type string The type of playlist, this will be set to spotify playlist for any imported playlists
name string The name of the playlist
description string The description of the playlist
songs an array of short song objects A list containing short song objects
creator string The id of the user who created the playlist
create_date integer The timestamp of when the playlist was created by a user
followers an array of strings A list of id's of the users that are currently following this playlist. This includes the current user.
artwork strings A URL for the cover art for the playlist. If none is supplied by the user, it will automatically generate one by using the first added track'S (that has a cover art) cover art
private boolean Has the owner made this playlist private? If set to false anyone can view this playlist and it can be returned as part of the browse endpoint
collaborative boolean Will be set to true if the owner has allowed other users to modify the playlist

id string The track id for the track
date_added integer The timestamp at which the track was added to the playlist
play_count integer The number of times the track has been played as part of the playlist
last_played integer The timestamp at which the track was last played as part of the playlist
