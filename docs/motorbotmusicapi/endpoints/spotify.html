Spotify

Endpoints for retrieving a users spotify information. This assumes that the user has connected their MotorBot Music account with their Spotify account.


Method Endpoint Usage Returns
GET /spotify/ Authenticate
GET /spotify/callback Authentication Callback
PUT /spotify/{spotify_playlist_id}/owner/{spotify_owner_id} Import Spotify Playlist playlist
GET /spotify/revoke Revoke Spotify Connection
PATCH /spotify/sync Toggle Playlist Syncing
