Overview

This documentation is split into two sections, one for the discordClient library and another for the motorbot music API.

This is the documentation for the MotorbotMusicAPI, an API used to communicate with motorbot to allow playback of media in a voice channel. You can switch in the top left drop down menu or select below if you'de like to change to the DiscordClient Library.

  • DiscordClient v0.1.0
  • MotorBot Music API v0.5.0

Important

This documentation is still under development and some information may not be complete yet. Monkey's are working on it!


Welcome

This is release 0.5.0 of the MotorbotMusicAPI which gives you access to the basic endpoints to allow the handling of music/media within Motorbot. This API is constantly being updated along with the documentation so check back regularly to look for any changes.
