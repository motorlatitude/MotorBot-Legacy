Overview

This documentation is split into two sections, one for the discordClient library and another for the motorbot music API.

This is the documentation for the discordClient Library, a nodeJS library used to create bots for discord guilds. You can switch in the top left drop down menu or select below if you'de like to change.

Welcome

This is release 0.1.0 of the discordClient Library which gives you access to all the available methods contained in the discord API (v6). This library is constantly updated along with the documentation so check back regularly to look for any changes.
