AudioPlayer
extends EventEmitter

Object that gets created on playing audio in a voice channel using playFromStream() or playFromFile() methods.

  • Javascript
  • CoffeeScript

channel.join().then(function(VoiceConnection){
    VoiceConnection.playFromFile("./file/path/to/audio.mp3").then(function(AudioPlayer){
       console.log(AudioPlayer)
    });
});
                

channel.join().then((VoiceConnection) ->
    VoiceConnection.playFromFile("./file/path/to/audio.mp3").then((AudioPlayer)->
       console.log AudioPlayer
    )
)

CONTENTS

Properties

A list of all available properties.

Property Type Description
glob_stream Stream The stream passed to the AudioPlayer
ffmpegDone Bool Is encoding finished for the audio stream
streamFinished Bool Has the audio finished, returns streamDone event
packageList Array Comma separated list of audio chunks
stopSend Bool Stop sending audio packets to the voiceHandler

Methods

A list of all available methods. 

  • pause()
    Pause the current stream. This keeps the audio chunks loaded and ready to be resumed once play() is called.
  • play()
    Play/resume the current stream.
  • stop()
    Stop the current stream. This deletes the stream and all loaded audio chunks, the stream cannot be resumed.
  • setVolume()
    Set current audio volume
  • getVolume()
    Get current audio volume

Events

A list of all events that can be fired by this object. 

  • ready
    Called once the AudioPlayer is ready to play the stream.
  • pause
    Called once the AudioPlayer has successfully paused the stream.
  • streamDone
    Called once the current stream has finished i.e. when the song has finished.
