DirectMessageChannel
Object

A Discord Direct Message Channel, which is normally a private text based channel used by users to private message either other. The Direct Message Channel object is similar to the TextChannel object with a few adjustments.

CONTENTS

Properties

A list of all available properties.

Property Type Description
client Object client object that was initially created
id String ID of channel
type Number Channel type, will usually be 1 for DM. 3 for group DM.
is_private Bool Is the channel private, this should normally be true for DM Channels
last_message_id String ID of the last message that was posted to this channel.
recipient Object User object containing of the DM recipient

  • getMessages(Object: options)
    Get messages in the channel

    Important

    The options, even though passed as an object, should only contain one of either; around, before or after. The limit is exempt from this and can be used alongside any of the three options.

    Parameter Type Optional Default Description 
    options
    		 Object 
    undefined
    		 Define which messages to retrieve relative to another message with a message_id which can be passed to either; String: around, String: before, String: after.
    An additional parameter exists to limit results: Number: limit (default 50)
    If no parameter is sent, the last 50 messages of a channel are returned.

    Examples

    • Javascript
    • CoffeeScript
    
var DiscordClient = require('DiscordClient');
var client = new DiscordClient({token: "TOKEN"});
// retrieve the last 20 messages before an incoming message
client.on("message",function(message){
    message.channel.getMessages({
        before: message.id,
        limit: 20
    }).then(function(messages){
        console.log(messages)
    }).catch(function(err){
        console.log(err.statusCode+" - "+err.statusMessage)
    })
});
client.connect();
