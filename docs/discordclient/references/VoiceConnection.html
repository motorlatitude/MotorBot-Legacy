VoiceConnection
extends EventEmitter

A VoiceConnection object, normally received when joining a VoiceChannel:

  • Javascript
  • CoffeeScript

channel.join().then(function(VoiceConnection){
   console.log(VoiceConnection);
});
                

channel.join().then((VoiceConnection) ->
    console.log VoiceConnection
