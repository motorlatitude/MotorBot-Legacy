Message
Object

A Discord Message from a Channel.


Properties

A list of all available properties.

Property Type Description
id String ID of message
channel_id String The channel id in which the message was posted
channel TextChannel The channels TextChannel object
guild_id String The guild id from the channel in which the message was posted
author User User object, containing the message author's details
content String The message content i.e. what the message said
timestamp Number Epoch timestamp when message was submitted
edited_timestamp Number Epoch timestamp of when the message was last edited, null for unmodified messages
tts Bool Does the message support Text-to-Speech (TTS)
mention_everyone Bool Does the message contain the @everyone tag
mentions Array A comma separated array containing User objects, for each mentioned user in a message using the @user format
mention_roles Array A comma separated array containing Role objects, for each mentioned user in a message using the @user format
attachments Array A comma separated array containing Attachment objects, for each attachment
embeds Array A comma separated array containing Embed objects, for each embed
reactions Array A comma separated array containing Reaction objects, for each reaction attached on a message
none String Used for validating a message was sent
pinned Bool Is the message pinned in a channel
webhook_id String If the message was generated by a webhook, this is the webhook's id

Methods

A list of all available methods.

Note

Currently not all of these methods are integrated into the library, only the addReaction method is available. The additional methods will be added in the near future.

Method Description 
addReaction:(emoji)
 Allows the addition of a reaction with a defined emoji to the message 
deleteReaction:(emoji)
 Allows the deletion of a reaction with a defined emoji from the message 
deleteUsersReaction:(emoji, userId)
 Allows the deletion of a reaction with a defined emoji added by a user with userId 
deleteAllReactions:()
 Removes all reactions from the message 
edit:(content)
 Edit the message, the new message is supplied via content 
delete:()
 Deletes the message permanently 
pin:()
 Pins message to channel 
deletePin:()
 Deletes the pinned message in a channel
