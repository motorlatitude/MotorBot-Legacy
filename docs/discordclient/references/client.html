client
extends EventEmitter

Object you create at the beginning of your bot.

Constructor

Create new client object, where "TOKEN" should be replaced with your API Token, supplied by discord. 


new DiscordClient({token: "TOKEN"});

Properties

A list of all available properties.

Property Type Description
guilds Array Comma separated array containing Guild objects
channels Array Comma separated array containing TextChannel/VoiceChannel objects
connected Bool Contains the state of the current gateway connection, true when connected.

Methods

A list of all available methods. 

  • connect()
    Establishes connection to discord gateway server. ready event is fired once connected.

Events

A list of all events that can be fired by this object.

Event Parameters Description 
ready
 
null
 Fired once connection to discord's gateway server has been successfully established. 
message
 
Message()
 Fired when a message was sent to one of the available channels, returns a Message object.
