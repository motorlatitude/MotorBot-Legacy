Overview

This documentation is split into two sections, one for the discordClient library and another for the motorbot music API.

This is the documentation for the discordClient Library, a nodeJS library used to create bots for discord guilds. You can switch in the top left drop down menu or select below if you'de like to change to the motorbot music api.

  • DiscordClient v0.1.0
  • MotorBot Music API v0.5.0
